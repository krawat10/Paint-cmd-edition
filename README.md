# Paint-cmd-edition
It is an image editor written in C. 
Functions:
-drawing rectangles (real-time view)
-drawing lines (real-time view)
-save/open .bmp files 
-save/open .xpm files 
-save/open own files
-support 16 colors
-undo function (basing on dynamic memory)
-open files by command
-open new files
